The Legato Application Framework started out as an initiative by Sierra Wireless Inc. to provide an open, secure and easy to use Application Framework to grow the "Internet of Everything". Legato Application Framework is designed to provide a smooth and connected open source platform that allows developers to easily work with the various wireless technologies and enable new ideas on IoT devices.
