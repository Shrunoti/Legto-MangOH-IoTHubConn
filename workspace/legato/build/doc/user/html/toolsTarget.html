Target Tools

Legato has these tools to run on the target:

Section Description
app list and control installed apps
cm control modem functions
config change config database
configEcm setup an ECM interface
fwupdate download image files directly to
inspect examine running Legato processes and memory pools
gnss monitor and debug GNSS
legato run Legato framework
log set logging variables for components
sbtrace help import files into sandboxed app
sdir control IPC bindings and troubleshoot
setNet set your MAC address or static IP
pmtool set power manager target options
uartMode set uart target options
update install, update, and remove software
wifi configure and control the WiFi service
