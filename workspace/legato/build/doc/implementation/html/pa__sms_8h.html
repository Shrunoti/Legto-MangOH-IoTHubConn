enum pa_sms_MsgType_t

Message Type Indicator. It is used for the message service configuration.

Enumerator
PA_SMS_DELIVER 

SMS-DELIVER (in the direction Service Center to Mobile station).

PA_SMS_SUBMIT 

SMS-SUBMIT (in the direction Mobile station to Service Center).

PA_SMS_STATUS_REPORT 

SMS-STATUS-REPORT.

PA_SMS_PDU 

PDU message.

PA_SMS_CELL_BROADCAST 

SMS Cell Broadcast.

PA_SMS_UNSUPPORTED 

SMS with unsupported format or encoding.

enum pa_sms_OptionMask_t

Option mask. It is used to know which option is present in the pa_sms_Message_t.

Enumerator
PA_SMS_OPTIONMASK_NO_OPTION 

No option.

PA_SMS_OPTIONMASK_OA 

TP Originating Address is present.

PA_SMS_OPTIONMASK_SCTS 

TP Service Centre Time Stamp is present.

PA_SMS_OPTIONMASK_DA 

TP Destination Address is present.

PA_SMS_OPTIONMASK_RA 

TP Recipient Address is present.
