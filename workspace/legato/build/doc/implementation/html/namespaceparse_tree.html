The calling function has found a bracketed var name at the location specified by 'begin'. Extract the name, stopping at a closing bracket. Throw an exception if an illegal character is found within the name.

Once the name is read, look up it's value and place this value into the output string.

Returns
Number of characters consumed in the original string.
Parameters
