void CheckForIMASigning ( mk::BuildParams_t &  buildParams)

Checks the info in buildParams object for IMA signing. If nothing specified in buildParams object then check environment variables for IMA signing and update buildParams object accordingly.

void FindToolChain ( mk::BuildParams_t &  buildParams)

Figure out what compiler, linker, etc. to use based on the target device type and store that info in the buildParams object.

If a given tool is not specified through the means documented in Tool Chain Configuration, then the corresponding entry in buildParams will be left empty.

void RunNinja ( const mk::BuildParams_t &  buildParams)

Run the Ninja build tool. Executes the build.ninja script in the root of the working directory tree, if it exists.

If the build.ninja file exists, this function will never return. If the build.ninja file doesn't exist, then this function WILL (quietly) return.

Exceptions
mktools::Exceptionif the build.ninja file exists but ninja can't be run.
