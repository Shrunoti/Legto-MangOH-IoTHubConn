Detailed Description

This file implements functions that can be used to import IMA keys (into the kernel keyring) and verify IMA signatures.

Copyright (C) Sierra Wireless Inc.

Macro Definition Documentation

#define EVMCTL_PATH   "/usr/bin/evmctl"

Path to evmctl tool. It can be used for producing and verifying IMA signatures. It can be also used to import keys into the kernel keyring.

le_result_t ima_ImportPublicCert ( const char *  certPath)

Import IMA public certificate to linux keyring. Public certificate must be signed by system private key to import it properly. Only privileged process with right permission and smack label will be able to do that.

Returns
  • LE_OK if imports properly
  • LE_FAULT if fails to import
