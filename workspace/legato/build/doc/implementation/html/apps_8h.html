void apps_Shutdown ( void  )

Initiates the shut down of all the applications. The shut down sequence happens asynchronously. A shut down handler should be set using apps_SetShutdownHandler() to be notified when all applications actually shut down.

le_result_t apps_SigChildHandler ( pid_t  pid)

The SIGCHLD handler for the applications. This should be called from the Supervisor's SIGCHILD handler.

Note
This function will reap the child if the child is an application process, otherwise the child will remain unreaped.
Returns
LE_OK if the signal was handled without incident. LE_NOT_FOUND if the pid is not an application process. The child will not be reaped. LE_FAULT if the signal indicates a failure of one of the applications which requires a system restart.
