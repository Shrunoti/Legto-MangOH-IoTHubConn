void envVars::SetTargetSpecific ( const mk::BuildParams_t &  buildParams)

Adds target-specific environment variables (e.g., LEGATO_TARGET) to the process's environment.

The environment will get inherited by any child processes, including the shell that is used to run the compiler and linker. Also, this allows these environment variables to be used in paths in .sdef, .adef, and .cdef files.
