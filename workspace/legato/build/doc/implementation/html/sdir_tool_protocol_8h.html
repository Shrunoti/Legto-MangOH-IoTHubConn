Enumeration Type Documentation

enum le_sdtp_MsgType_t

Message type IDs.

Enumerator
LE_SDTP_MSGID_LIST 

List all contents of the Service Directory. Payload is a file descriptor to which output should be written.

LE_SDTP_MSGID_LIST_JSON 

Same as LE_SDTP_MSGID_LIST, but the output in json format.

LE_SDTP_MSGID_UNBIND_ALL 

Delete all bindings (This message has no payload).

LE_SDTP_MSGID_BIND 

Create one binding. The payload is the binding details. If the Service Directory runs into an error, it will drop the connection to the sdir tool without responding.
