Detailed Description

File Descriptor Monitor

These keep track of file descriptors that are being monitored by a particular thread. They are allocated from a per-thread FD Monitor Sub-Pool and are kept on the thread's FD Monitor List. In addition, each has a Safe Reference created from the FD Monitor Reference Map.
