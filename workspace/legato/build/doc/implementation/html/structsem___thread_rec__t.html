Detailed Description

Semaphore Thread Record.

This structure is to be stored as a member in each Thread object. The function thread_GetSemaphoreRecPtr() is used by the semaphore module to fetch a pointer to one of these records for a given thread.

Warning
No code outside of the semaphore module (semaphore.c) should ever access the members of this structure.
