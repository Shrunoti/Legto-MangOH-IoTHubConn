void ninja::AppBuildScriptGenerator_t::GenerateStagingBundleBuildStatements ( model::App_t *  appPtr)
protectedvirtual

Write to a given build script the build statements for bundling a given app's files into the app's staging area.

Note
Uses a set to track the bundled objects (destination paths) that have been included so far. This allows us to avoid bundling two files into the same location in the staging area. The set can also be used later by the calling function to add these staged files to the bundle's dependency list.
