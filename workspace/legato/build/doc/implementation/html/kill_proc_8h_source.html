26 /**
27  * Initiates a soft kill of a process. This sends the process a SIGTERM signal allowing the process
28  * to catch the signal and perform a graceful shutdown. If the process fails to shut down within
29  * the timeout period a hard kill (SIGKILL) will be performed. If the calling process knows that
30  * the target process has already died it can call kill_Died() to cancel the hard kill timeout. The
31  * calling process must have privileges to send signals to the process specified by pid.
32  *
33  * If the process does not exist this function simply returns.
34  *
35  * @note
36  * Does not return on error.
37  */
