Detailed Description

Structure used to keep track of the trees loaded in the configTree daemon.

Field Documentation

ssize_t Tree_t::activeReadCount

Count of reads that are currently active on this tree.

ni_IteratorRef_t Tree_t::activeWriteIterRef

The parent write iterator that's active on this tree. NULL if there are no writes pending.

bool Tree_t::isDeletePending

If this is set to true, then the tree will be deleted once the last iterator has been closed on it. If it is set to false, the tree is left alone.
