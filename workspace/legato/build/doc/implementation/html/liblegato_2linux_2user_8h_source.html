28 /**
29  * Creates a user account with the specified name. A group with the same name as the username will
30  * also be created and the group will be set as the user's primary group. If the user and group are
31  * successfully created the user ID and group ID are stored at the location pointed to by uidPtr and
32  * gidPtr respectively. If the user account already exists then LE_DUPLICATE will be returned and
33  * the user account's user ID and group ID are stored at the location pointed to by uidPtr and
34  * gidPtr respectively. If there is an error then LE_FAULT will be returned and the values at
35  * uidPtr and gidPtr are undefined.
36  *
37  * @return
38  * LE_OK if successful.
39  * LE_DUPLICATE if the user or group already exists.
40  * LE_FAULT if there was an error.
41  */
42 //--------------------------------------------------------------------------------------------------
