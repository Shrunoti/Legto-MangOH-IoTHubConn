Detailed Description

Implements the "Interface" objects and the "Interface List" in the Legato Low-level Messaging API.

See messaging.c for the overall subsystem design.

Warning
The code in this file must be thread safe and re-entrant.

Copyright (C) Sierra Wireless Inc.
