Detailed Description

Inter-module definitions exported by the Session module of the Low-level Messaging API implementation.

See messaging.c for an overview of the Low-level Messaging API implementation.

Enumeration Type Documentation

enum msgSession_SessionState_t

Enumerates all the possible states that a Session object can be in.

Enumerator
LE_MSG_SESSION_STATE_CLOSED 

Session is closed.

LE_MSG_SESSION_STATE_OPENING 

Client is trying to open the session. Waiting for the server's response. (Note: This is a client-only state.)

LE_MSG_SESSION_STATE_OPEN 

Session is open.
