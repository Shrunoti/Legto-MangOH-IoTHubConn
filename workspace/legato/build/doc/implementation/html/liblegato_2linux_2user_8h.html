le_result_t user_Create ( const char *  usernamePtr,
uid_t *  uidPtr,
gid_t *  gidPtr 
)

Creates a user account with the specified name. A group with the same name as the username will also be created and the group will be set as the user's primary group. If the user and group are successfully created the user ID and group ID are stored at the location pointed to by uidPtr and gidPtr respectively. If the user account already exists then LE_DUPLICATE will be returned and the user account's user ID and group ID are stored at the location pointed to by uidPtr and gidPtr respectively. If there is an error then LE_FAULT will be returned and the values at uidPtr and gidPtr are undefined.

Returns
LE_OK if successful. LE_DUPLICATE if the user or group already exists. LE_FAULT if there was an error.

Creates a user account with the specified name. A group with the same name as the username will also be created and the group will be set as the user's primary group. If the user/group accounts already exist then LE_DUPLICATE will be returned and the user account's user ID and group ID are stored at the location pointed to by uidPtr and gidPtr respectively. If there is an error then LE_FAULT will be returned and the values at uidPtr and gidPtr are undefined.
