Detailed Description

Stored data by legato File Systm(FS) service may require new treatment after new system is installed. This file implements few api to check/update whether new system is installed.

le_result_t fsSys_FlagNewSys ( void  )

After installation of new legato, stored data by file system service (i.e. le_fs* api) may require new treatment. This function flags that a new system is installed.

Returns
  • LE_OK The function succeeded.
  • LE_NOT_PERMITTED Access denied to the file containing new system flag.
  • LE_FAULT The function failed.
