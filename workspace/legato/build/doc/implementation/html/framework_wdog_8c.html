Detailed Description

Implements the frameworkWdog.api by setting up a timer in a framework daemon to notify watchdog that the daemon is still alive.

Framework daemons cannot use the normal wdogChain API since the watchdog relies on most of the other framework daemons for its operation. Having the daemons as clients of the watchdog would introduce a circular dependency, leading to eventual deadlocks.
