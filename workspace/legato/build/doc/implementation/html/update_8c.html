Detailed Description

Client of update service provider(i.e. UpdateDaemon) for installing/removing app, and installing firmware. This client receives update package via STDIN and sequentially calls update APIs for successful update. This client follows the steps mentioned in le_update.api documentation while performing update. A callback is implemented here to receive progress of ongoing update process.

Tell the argument parser to allow less positional arguments than positional callbacks.
