Detailed Description

Timer Thread Record.

This structure is to be stored as a member in each Thread object. The timer module uses the function thread_GetTImerRecPtr() to fetch a pointer to one of these records for a given thread.

Warning
No code outside of the timer module (timer.c) should ever access members of this structure.
