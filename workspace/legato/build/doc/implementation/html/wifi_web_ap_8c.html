#define LOGFILE   "/tmp/wifi_http.log"

This module implements a Web interface for setting up the WiFi access point

Copyright (C) Sierra Wireless Inc.

It starts the webserver and it subscribes to the WiFi access point events and logs them to a logfile. The page index.html provides an interface page with two parts. The first part is an interface to setup the WiFi access point. The second part is the content of the logfile provided by this app.
