Parses a file. Calls a provided section parser function for each section found in the file.

The section parser function must return a pointer to a section (CompoundItem_t, which will be added to the list of sections in the DefFile_t), or throw an exception on error.

Exceptions
mk::Exception_tif an error is encountered.
