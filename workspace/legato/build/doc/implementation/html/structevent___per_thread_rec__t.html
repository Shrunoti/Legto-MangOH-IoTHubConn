Detailed Description

Event Loop's per-thread record.

One of these must be allocated as a member of the Thread object. The Event Loop module will call the function thread_GetEventRecPtr() to fetch a pointer to it.

Warning
No code outside of the Event Loop module or the FD Monitor module should ever access any member of this structure.
