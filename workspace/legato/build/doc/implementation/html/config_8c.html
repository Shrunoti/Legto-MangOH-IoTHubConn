This function will attempt read a value from the tree, and write it to standard out, or to a file. The tree data will be written in JSON format.

If the specified node is a stem, then the tree structure will be dumped, starting at the specified node. If a '*' is given for a node path then all trees in the system will be dumped into a JSON document.
