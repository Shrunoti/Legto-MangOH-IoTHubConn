le_result_t addr_GetLibDataSection ( pid_t  pid,
const char *  libNamePtr,
off_t *  libAddressPtr 
)

Gets the address of the .data section of the specified library for the specified process. The address is in the address space of the specified process.

Returns
LE_OK if successful. LE_NOT_FOUND if the library was not found. LE_FAULT if there was an error.
Parameters
[in]pidPid to get the address for. Zero means the calling process.
[in]libNamePtrName of the library.
[out]libAddressPtrAddress of the .data section of the library.
