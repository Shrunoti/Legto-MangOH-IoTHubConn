Initiates a soft kill of a process. This sends the process a S\+I\+G\+T\+E\+RM signal allowing the process to catch the signal and perform a graceful shutdown. If the process fails to shut down within the timeout period a hard kill (S\+I\+G\+K\+I\+LL) will be performed. If the calling process knows that the target process has already died it can call \hyperlink{kill_proc_8c_aaf56ee79805a6ac10670ea67a57b0cea}{kill\+\_\+\+Died()} to cancel the hard kill timeout. The calling process must have privileges to send signals to the process specified by pid.

If the process does not exist this function simply returns.

\begin{DoxyNote}{Note}
Does not return on error. 
\end{DoxyNote}
