\subsection{Detailed Description}
Implements the framework\+Wdog.\+api by setting up a timer in a framework daemon to notify watchdog that the daemon is still alive.

Framework daemons cannot use the normal wdog\+Chain A\+PI since the watchdog relies on most of the other framework daemons for its operation. Having the daemons as clients of the watchdog would introduce a circular dependency, leading to eventual deadlocks.
