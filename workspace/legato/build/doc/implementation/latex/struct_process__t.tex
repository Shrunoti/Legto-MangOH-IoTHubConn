\subsection{Detailed Description}
Process class. 

The process object.

Definition of the Process class, which is used to hold the details of a single process defined in a \char`\"{}run\+:\char`\"{} subsection of a \char`\"{}processes\+:\char`\"{} section in a .adef file.
