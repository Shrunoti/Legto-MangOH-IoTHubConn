Close an iterator object and invalidate it\textquotesingle{}s external safe reference. (If there is one.) Once done, this iterator is no longer accessable from outside of the process.

An iterator is closed with out releasing in the case where you have an open write iterator on a tree with open reads. The iterator is closed, but it\textquotesingle{}s data is not yet mergable into the tree.

So, the iterator is marked as closed and it\textquotesingle{}s external ref is invalidated so no more work can be done with that iterator.
