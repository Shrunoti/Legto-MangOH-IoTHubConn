\subsection{Detailed Description}
Lexical Analyzer (Lexer) for the mk$\ast$ tools.

The Parsers use the \hyperlink{class_lexer__t}{Lexer\+\_\+t} to get lexical tokens from the .Xdef input file.

As a side-\/effect, the \hyperlink{class_lexer__t}{Lexer\+\_\+t} builds a list of tokens in a given \hyperlink{struct_def_file__t}{Def\+File\+\_\+t} object.
