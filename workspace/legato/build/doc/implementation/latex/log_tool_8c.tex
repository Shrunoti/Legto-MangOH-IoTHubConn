\subsection{Detailed Description}
This is the Legato log command line tool used to set logging configurations for legato components. When the user executes the log command the proper arguments must be specified depending on the command. The command is then translated and sent to the Legato log daemon which forwards it to the correct destination.

The general format of log commands is\+:

\begin{DoxyVerb}$ log command commandParameter destination
\end{DoxyVerb}


The following are examples of supported commands\+:

To set the log level to I\+N\+FO for a component in a process\+: \begin{DoxyVerb}$ log level INFO processName/componentName
\end{DoxyVerb}


To enable a trace\+: \begin{DoxyVerb}$ log trace keyword processName/componentName
\end{DoxyVerb}


To disable a trace\+: \begin{DoxyVerb}$ log stoptrace keyword processName/componentName
\end{DoxyVerb}


With all of the above examples \char`\"{}$\ast$\char`\"{} can be used in place of process\+Name and component\+Name to mean all processes and/or all components. In fact if the \char`\"{}process\+Name/component\+Name\char`\"{} is omitted the default destination is set to all processes and all components.

The translated command to send to the log daemon has this format\+:

\begin{DoxyVerb} *    ----------------------------------------
 *    | cmd | destination | commandParameter |
 *    ----------------------------------------
 * \end{DoxyVerb}


where, cmd is a command code that is one byte in length. destination is the \char`\"{}process\+Name/component\+Name\char`\"{} followed by a \textquotesingle{}/\textquotesingle{} character. command\+Parameter is the string specific to the command.
