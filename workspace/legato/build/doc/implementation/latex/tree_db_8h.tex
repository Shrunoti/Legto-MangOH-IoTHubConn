\subsection{Detailed Description}
Interface to the low level tree DB structure.

There are basically two types of tree\+:
\begin{DoxyItemize}
\item Named trees
\item Shadow trees
\end{DoxyItemize}

Both are accessed the same basic way\+:
\begin{DoxyEnumerate}
\item Get a reference to the tree.
\item Get a reference to the root node of the tree.
\item Traverse the tree.
\item Read/write the node values and/or create/delete new child nodes.
\end{DoxyEnumerate}

The shadow trees are used to support write transactions. When a write transaction is started for a given named tree, a shadow tree is created to \char`\"{}shadow\char`\"{} that named tree. All changes to the tree are only done through the shadow tree. When the write transaction is committed, the shadow tree is \char`\"{}merged\char`\"{} back into the named tree and then the shadow tree is deleted. To cancel a write transaction, just delete the shadow tree without merging it back into the named tree.
