\subsection{Detailed Description}
This is the process class that is used to reference the Supervisor\textquotesingle{}s child processes in applications. This class has methods for starting and stopping processes and keeping process state information. However, a processes state must be updated by calling the \hyperlink{proc_8c_ac8b7e877a595bfa590b21d10023459ea}{proc\+\_\+\+Sig\+Child\+Handler()} from within a S\+I\+G\+C\+H\+I\+LD handler.
