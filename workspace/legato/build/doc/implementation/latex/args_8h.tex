\subsection{Detailed Description}
Command-\/\+Line Argument Processing Support

The command line processing consists of two phases. In the first phase, all expected arguments are registered with the args scanner. Finally, the arguments are scanned by a call to \hyperlink{namespaceargs_aaa94887be475da49f06e7b539f2ee8b2}{Scan()}.

Example\+:


\begin{DoxyCode}
\textcolor{keywordtype}{void} MyLooseArgHandler(\textcolor{keywordtype}{void}* context, \textcolor{keyword}{const} \textcolor{keywordtype}{char}* argValueString)
\{
    \textcolor{comment}{// Add argument string to list of things to be processed.}
    ...
\}

\textcolor{keywordtype}{int} \hyperlink{start_8c_a3c04138a5bfe5d72780bb7e82a18e627}{main}(\textcolor{keywordtype}{int} argc, \textcolor{keywordtype}{char}** argv)
\{
    \textcolor{keywordtype}{bool} isExtreme = \textcolor{keyword}{false};

    \hyperlink{namespaceargs_ad246cd5717a600f256fe3d3dbb4dcba9}{args::AddOptionalFlag}(&isExtreme, \textcolor{stringliteral}{"x"}, \textcolor{stringliteral}{"extreme"}, \textcolor{stringliteral}{"Take it to the limit?"});
    \hyperlink{namespaceargs_a7a76d0392a64edfae4a1f918ceaebd53}{args::SetLooseArgHandler}(MyLooseArgHandler, NULL);

    \hyperlink{namespaceargs_aaa94887be475da49f06e7b539f2ee8b2}{args::Scan}();

    \textcolor{keywordflow}{if} (isExtreme)
    \{
        \textcolor{comment}{// Do something extreme!}
        ...
    \}
    \textcolor{comment}{// Process list of things received as arguments.}
    ...
\}
\end{DoxyCode}
