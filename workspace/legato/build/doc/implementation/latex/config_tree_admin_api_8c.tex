Read a subset of the configuration tree from the given file\+Path. That tree then overwrites the node at the given node\+Path.

This function will import a sub-\/tree as part of the iterator\textquotesingle{}s current transaction. This allows you to create an iterator on a given node. Import a sub-\/tree, and then examine the contents of the import before deciding to commit the new data.

{\bfseries Responds} {\bfseries With\+:} 

Responds with one of the following values\+: \begin{DoxyVerb}    - LE_OK            - Commit was completed successfully.
    - LE_FAULT         - An I/O error occurred while reading the data.
    - LE_FORMAT_ERROR  - Configuration data being imported appears corrupted.\end{DoxyVerb}
